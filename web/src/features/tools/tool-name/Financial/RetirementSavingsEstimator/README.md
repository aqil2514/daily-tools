# Retirement Savings Estimator

Alat untuk memperkirakan berapa besar tabungan bulanan yang dibutuhkan agar dapat mencapai target dana pensiun berdasarkan usia, tabungan saat ini, dan estimasi return tahunan.

## Tujuan
Tool ini dirancang sebagai **alat estimasi dan edukasi**, bukan sebagai alat perencanaan atau nasihat keuangan profesional.

Fokus utama:
- Memberikan gambaran kasar kebutuhan tabungan pensiun
- Membantu pengguna memahami hubungan antara waktu, tabungan, dan pertumbuhan dana

## Input
- Usia saat ini
- Usia target pensiun
- Total tabungan saat ini
- Target dana pensiun
- Estimasi return tahunan (berdasarkan asumsi pengguna)

## Output
- Estimasi tabungan bulanan yang dibutuhkan
- Waktu menuju usia pensiun (tahun & bulan)
- Nilai estimasi tabungan saat usia pensiun
- Sisa target dana yang perlu dicapai

## Catatan
- Semua perhitungan dilakukan langsung di browser (tanpa server).
- Hasil yang ditampilkan bersifat estimasi berdasarkan asumsi pengguna.
- Tool ini **tidak memperhitungkan** inflasi, pajak, alokasi aset, atau perubahan gaya hidup (secara desain).
- Tool ini **bukan** pengganti perencanaan keuangan profesional.
